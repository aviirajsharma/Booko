# 📚 Booko - Download & Read Free Books

Booko is an Android app that lets users discover, download, manage, and read free PDF books with a modern Jetpack Compose UI.

## Features
- Browse and download free PDF books
- Read books offline
- Manage downloaded books locally
- Real-time library updates with Flow
- Smooth navigation using Compose Navigation

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM + Clean Architecture
- Room Database
- Coroutines & Flow
- Hilt
- Compose Navigation

## Architecture
The app follows **MVVM + Clean Architecture** for better scalability, maintainability, and lifecycle-aware state management.
